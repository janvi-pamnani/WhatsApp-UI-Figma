# WhatsApp-UI-Figma
WhatsApp UI Clone designed in Figma

# Project Overview:
This project is a UI/UX design of a WhatsApp-inspired mobile chat application created using Figma.
It demonstrates modern interface design, user flow, and interactive prototyping.

# Key Features:
1.Chat list screen with user previews
* Conversation interface with message layout
* Clean and minimal UI design
* Interactive navigation between screens

# Tools Used: Figma
